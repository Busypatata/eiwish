# EiWish

Make wishing easy. Create and organize your wishlists, share them with the people who matter, and let everyone know exactly what you want — without the guesswork.

## Features

- Create unlimited wishlists
- Private, shared, or public visibility
- Add wishes with photos, price, and links
- Invite collaborators by username
- Mark items as bought to avoid duplicate gifts
- Public profile page for every user

## Tech Stack

Next.js · TypeScript · Tailwind CSS · Supabase · Vercel
